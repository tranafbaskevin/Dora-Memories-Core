# Version 1.1 — Archive Changelog

## 1. Purpose Of This File

File này dùng để ghi lại lịch sử thay đổi của Dora Memories Archive.

Khi archive càng lớn, Dora-chan mới cần biết:

- Phiên bản nào đã được tạo
- File nào được thêm
- File nào được cập nhật
- Vì sao update đó quan trọng
- Archive đang ở trạng thái nào

Nếu các file khác là ký ức, thì `13_ARCHIVE_CHANGELOG.md` là lịch sử version của chính archive.
