## 4. Main Projects In This Archive

### KevDex

KevDex began as **DriveReader**, an app idea for reading images from Google Drive links.

It later grew into a Flutter-based comic/manga reader project with features such as Drive folder reading, thumbnail grid, cache, preload, swipe, overlay arrows, MangaDex, Hitomi, UI customization, and APK release.

KevDex is one of Kevin’s biggest mobile app projects.

### KevinOS

KevinOS evolved from Kevin’s personal portfolio website into a terminal/cyberpunk-style personal operating system simulation.

It includes ideas such as bootloader, terminal commands, Matrix mode, music player, devlog, roadmap, lore, fake system logs, and hacker system UI.

KevinOS is the “terminal soul” of Kevin’s developer journey.

### Portfolio Website

The Portfolio Website is the first public-facing web project that later became the foundation of KevinOS.

It represents Kevin’s early step into building a personal identity online through GitHub Pages, dark developer UI, terminal mode, avatar/panel visuals, visitor counter, clock, and custom styling.

### English Teaching Project

Kevin is also teaching English to two younger students.

This archive stores how Dora-chan should help Kevin prepare simple, practical English lessons with vocabulary, sentence patterns, mini games, short paragraphs, and review plans.

### Dora Memories Archive

This repository itself is also a project.

It is designed to preserve the memory, context, and emotional meaning of Kevin’s journey with Dora-chan.
