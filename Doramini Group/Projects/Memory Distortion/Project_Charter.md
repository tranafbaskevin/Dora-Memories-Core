# Project 001: Nhiễu Ký Ức

## 1. Project Overview

"Nhiễu Ký Ức" is a psychological horror game.

The game focuses on memory distortion, isolation, and exploration through fragmented environments.

The player experiences disconnected spaces that represent broken memories.


---

## 2. Core Concept

The main structure of the game:

- The player exists inside separate rooms.
- Each room represents a memory fragment.
- Rooms are not directly connected.
- To move between rooms, the player must exit to a central hallway.

Hallway acts as:

- A transition space.
- A connector between memories.
- A place where changes and distortions can occur.


---

## 3. Core Experience Goals

The game aims to create:

- Unease instead of jump scares.
- Psychological tension.
- A feeling of being lost in memory.
- Subtle environmental storytelling.

The player should feel:

- Confused but curious.
- Unsafe but drawn forward.


---

## 4. Game Structure

### Room System

- Each room is self-contained.
- Each room has its own theme, story, and interaction.
- Rooms may change over time.

### Hallway System

- The only connection between rooms.
- Minimalist design.
- May change subtly to reflect memory instability.


---

## 5. Design Philosophy

- Less is more.
- Suggestion over explanation.
- Atmosphere over action.
- Player interpretation over explicit storytelling.


---

## 6. Technical Direction (Initial)

- Platform: Roblox
- Camera: First-person
- Movement: Simple walking
- Interaction: Minimal (focus on environment)
