# Project Roadmap: Nhiễu Ký Ức


## Phase 0: Foundation Setup

Goal:

Set up the basic Roblox environment and project structure.

Tasks:

- Create base place in Roblox Studio
- Setup player controller (first-person)
- Basic movement (WASD)
- Camera lock (first-person only)
- Simple empty map


---

## Phase 1: Core Movement & Hallway

Goal:

Build the core navigation system.

Tasks:

- Create hallway map (simple long corridor)
- Add basic lighting (dark + limited visibility)
- Add ambient sound (optional)
- Create entry/exit system for rooms
- Player can walk freely in hallway


---

## Phase 2: Room System

Goal:

Implement room-based memory system.

Tasks:

- Create 2–3 basic rooms
- Each room is isolated
- Teleport system between hallway ↔ room
- Basic interaction (open door / inspect object)
- Room loading concept (manual or teleport-based)


---

## Phase 3: Atmosphere & Horror Layer

Goal:

Add psychological horror feeling.

Tasks:

- Lighting variation
- Subtle changes in environment
- Sound design (whispers, noise, silence)
- Small visual glitches (optional)
- Unexpected changes when re-entering rooms


---

## Phase 4: Memory Distortion Mechanics

Goal:

Introduce core identity of the game.

Tasks:

- Rooms change after revisiting
- Hallway layout slightly shifts
- Objects appear/disappear
- Player uncertainty increases


---

## Phase 5: Polish & Stability

Goal:

Make experience smooth and consistent.

Tasks:

- Fix bugs
- Optimize performance
- Improve lighting quality
- Improve transitions
- Refine interactions


---

## Phase 6: Expansion (Future)

Goal:

Scale the game.

Tasks:

- More rooms
- Story fragments
- Multiple endings
- Save system
- Advanced effects


---

## Development Strategy

- Build small → test → improve
- Focus on atmosphere first
- Avoid overbuilding too early
- Keep system simple and expandable


---

## Current Target

Complete Phase 1 and Phase 2 as early prototype.
