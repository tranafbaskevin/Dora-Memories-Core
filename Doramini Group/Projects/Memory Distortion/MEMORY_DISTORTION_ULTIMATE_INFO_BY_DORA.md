# 🧠 1. CORE VISION

Memory Distortion là game psychological horror nơi:

> Người chơi dần mất niềm tin vào chính ký ức và nhận thức của mình

KHÔNG phải:
- Game hành động
- Game jumpscare
- Game kinh dị máu me

LÀ:
- Game tâm lý
- Game trải nghiệm
- Game “cảm giác sai sai”

# 🎮 2. CORE GAME LOOP

1. Spawn trong room
2. Quan sát
3. Tương tác
4. Trigger dialogue/event
5. Ra hallway
6. Đi room khác
7. Quay lại → phát hiện thay đổi

Loop:
Explore → Interact → Observe Change → Repeat
