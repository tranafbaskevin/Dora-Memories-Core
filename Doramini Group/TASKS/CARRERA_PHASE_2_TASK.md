# Carrera Task - Phase 2 Room & House System

Project:
Memory Distortion

Developer:
Carrera (Antigravity)

Phase:
Phase 2 - Room & House System


## Objective

Xây dựng hệ thống không gian nhà cho Memory Distortion.

Phase này tập trung vào architecture, chưa cần hoàn thiện art.


## Important Design Rules

Memory Distortion không phải game FPS.

Phong cách:

- Top-down
- RPG Maker inspired
- Visual Novel hybrid


Full house layout chỉ dùng làm reference.

Gameplay thực tế:

- Player di chuyển trong hallway.
- Mỗi phòng là một scene riêng.
- Camera giới hạn trong từng khu vực.
